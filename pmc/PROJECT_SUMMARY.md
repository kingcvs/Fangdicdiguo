# 房地产帝国 - 项目总结

## 📋 项目概述
这是一个硬核的房地产公司模拟经营游戏，使用原生HTML5/CSS3/JavaScript开发。

### 项目详情页面功能
1. **项目头部信息**
   - 项目名称、等级标签（高端/改善/刚需）
   - 项目状态标签（报批）
   - 项目信息（城市、建面、类型、销售方式）
   - 8个进度阶段（四证、设计、施工、预售、结算、竣工、交付、完成）

2. **项目经理信息**
   - 头像、姓名、性格标签
   - 星级评价
   - 月薪显示
   - 加成效果（质量、成本、速度、销售）
   - 描述信息
   - 自动模式开关和解雇按钮

3. **下一步提示**
   - 引导式的下一步操作提示
   - 前往按钮

4. **7个功能标签页**
   - 四证办理
   - 设计管理
   - 工程管理
   - 成本控制
   - 招采管理
   - 单元规划
   - 资金管理

5. **四证办理模块**
   - 公司可用现金显示
   - 办理流程说明
   - 四个证书的状态（待办理/办理中/已完成）
   - 申请办理按钮

6. **设计管理模块**
   - 当前阶段显示
   - 项目定位选择（刚需/改善/高端）
   - 外立面风格选择（现代简约/新中式/ArtDeco/法式）
   - 成本、售价、需求系数
   - 规划指标（容积率）

## 🚀 如何部署项目

### 本地运行
项目可以直接在浏览器中打开 `index.html` 运行，或者使用任何静态文件服务器：

```bash
# 使用Python
python -m http.server 8000

# 使用Node.js (http-server)
npx http-server -p 8000

# 或者直接在浏览器中打开 index.html
```

### 部署到云端

#### 选项1: GitHub Pages
1. 创建GitHub仓库
2. 推送代码到仓库
3. 在仓库设置中开启GitHub Pages

#### 选项2: Vercel / Netlify
1. 导入GitHub仓库
2. 自动部署（这两个平台都支持静态站点）

#### 选项3: 传统云服务器
1. 上传所有文件到服务器
2. 使用Nginx/Apache等Web服务器托管
